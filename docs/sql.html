SQL-запити

На цьому етапі проєкту було реалізовано низку SQL-запитів для роботи з базою даних. Запити охоплюють основні операції — створення таблиць, додавання записів, оновлення, видалення та вибірку даних.

Створення таблиць


CREATE TABLE Students (
  id INT PRIMARY KEY,
  name VARCHAR(100),
  group_name VARCHAR(50)
);

Вставка даних


INSERT INTO Students (id, name, group_name)
VALUES (1, 'Іваненко Ірина', 'ІО-21');

Запити SELECT


SELECT * FROM Students;

SELECT name FROM Students WHERE group_name = 'ІО-21';

Оновлення та видалення


UPDATE Students SET name = 'Петренко Олег' WHERE id = 1;

DELETE FROM Students WHERE id = 1;

SQL-запити дозволяють ефективно керувати базою даних і отримувати потрібну інформацію відповідно до поставлених задач. Наведені приклади демонструють базовий функціонал, реалізований у рамках проєкту.

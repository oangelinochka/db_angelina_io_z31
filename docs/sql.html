SQL-запити

Приклади SQL-запитів, що використовуються в базі даних:

CREATE TABLE Student (
  id INT PRIMARY KEY,
  name VARCHAR(100)
);

CREATE TABLE Subject (
  id INT PRIMARY KEY,
  title VARCHAR(100)
);

Приклади SQL-запитів

Створення таблиці projects

CREATE TABLE projects (
  project_id SERIAL PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  description TEXT,
  start_date DATE,
  end_date DATE
);

Отримати всі задачі по проєкту

SELECT t.task_id, t.description, t.status, u.name AS executor
FROM tasks t
JOIN users u ON t.user_id = u.user_id
WHERE t.project_id = 1;

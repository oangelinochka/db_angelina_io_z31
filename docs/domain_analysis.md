
# Аналіз предметної області

Система управління проектами призначена для організації, відстеження та керування завданнями, проектами та командами.

Основні учасники:
- Керівник проекту
- Члени команди
- Замовники

Основні об'єкти:
- Проєкти
- Завдання
- Користувачі
- Коментарі
- Статуси

Ключові процеси:
- Створення проєкту
- Призначення учасників
- Постановка завдань
- Відслідковування прогресу

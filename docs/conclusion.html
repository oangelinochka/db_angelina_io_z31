Висновки

У ході виконання проєкту було створено концептуальну ER-модель бази даних, реалізовано SQL-запити для взаємодії з таблицями та оформлено документацію у вигляді вебсайту.

Проєкт демонструє володіння базовими навичками в області проєктування баз даних, нормалізації даних, побудови зв’язків між сутностями та застосування SQL для створення, заповнення та обробки даних.

Створений сайт є зручною формою представлення документації й може бути корисним прикладом для подальших подібних робіт.

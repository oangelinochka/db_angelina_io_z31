
# Модель бізнес-об'єктів

**Проєкт**
- id
- назва
- опис
- дата початку
- дата завершення

**Користувач**
- id
- ім'я
- email
- роль

**Завдання**
- id
- назва
- опис
- виконавець
- статус
- дедлайн

**Коментар**
- id
- текст
- дата
- автор
- завдання

Зв'язки:
- Один проєкт має багато завдань
- Один користувач може мати багато завдань
- Завдання може мати багато коментарів

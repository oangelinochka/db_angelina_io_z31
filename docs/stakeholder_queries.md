
# Запити зацікавлених осіб

**Керівник проекту хоче:**
- Створити новий проєкт
- Додати учасників
- Переглядати статуси завдань

**Член команди хоче:**
- Переглянути призначені йому завдання
- Оновити статус виконання
- Додати коментар до завдання

**Замовник хоче:**
- Переглянути загальний прогрес проєкту

Опис RESTful API

RESTful сервіс надає інтерфейс для роботи з ресурсами системи управління проєктами.

  • GET /projects — отримати список проєктів
  • GET /projects/{id} — отримати інформацію про конкретний проєкт
  • POST /tasks — створити нову задачу
  • PUT /tasks/{id} — оновити задачу
  • DELETE /tasks/{id} — видалити задачу

Приклад запиту

GET /projects HTTP/1.1
Host: api.projectmanager.com
Accept: application/json

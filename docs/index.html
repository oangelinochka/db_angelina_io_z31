Вступ

Цей проєкт присвячений створенню бази даних для конкретної предметної області. У роботі було розроблено ER-модель, яка відображає основні сутності та зв’язки між ними. На основі моделі створено логічну структуру бази даних із таблицями, ключами та обмеженнями цілісності. Для роботи з базою використано мову SQL — написані запити для додавання, оновлення, видалення та отримання даних. Документація включає опис моделі, приклади SQL-запитів та висновки. Цей проєкт демонструє практичні навички моделювання, нормалізації та роботи з реляційними базами даних.
